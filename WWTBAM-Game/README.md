# Millionaire Game

Игра "Кто хочет стать миллионером?", реализованная на SwiftUI.

## 🧩 Особенности

- ✅ 15 вопросов в каждой игре (5 легких, 5 средних, 5 сложных)
- ✅ 3 подсказки:
  - **50/50** — убирает два неправильных варианта
  - **Звонок другу** — имитация совета
  - **Помощь зала** — отображение предполагаемого выбора
- ✅ Несгораемые суммы
- ✅ Поддержка таймера на ответ
- ✅ Финальный экран с отображением суммы и уровня
- ✅ Сохранение лучшего выигрыша
- ✅ Логика и UI полностью разделены (архитектура MVVM)

## Запуск
Открой `MillionaireGame.xcodeproj` или `.xcworkspace` в Xcode и нажми `Cmd + R`.

## 📲 Пользовательский сценарий

### 🪄 Запуск приложения
- При запуске отображается **Launch Screen**
- Далее пользователь попадает на **главный экран**
- Если есть сохранённая игра — отображается кнопка **«Продолжить»**
- В правом верхнем углу — кнопка **«Правила игры»**

### 🎮 Экран игры
- Отображается **вопрос и 4 варианта ответа**
- Доступны 3 подсказки:
  - `50/50`
  - `Помощь зала`
  - `Звонок другу`
- Также есть кнопка для **просмотра таблицы выигрышей**

![](Game.png)
![](table.png)

### ⌛ Обработка ответа
- После выбора ответа — **ожидание 5 секунд**
- В это время играет **интригующая музыка**
- Затем:
  - Если ответ **правильный**:
    - Подсвечивается текущий ответ **зелёным**
    - Переход на экран со списком 15 уровней
    - Подсвечивается текущий вопрос **зелёным**
    - Воспроизводится **музыка победы**
  - Если ответ **неверный**:
    - Подсвечивается текущий ответ **красным**
    - Переход на экран со списком уровней
    - Подсвечиваются несгораемые суммы **синим**
    - Показывается сумма, либо **«Вы выиграли 0»**
    - Воспроизводится **музыка проигрыша**

![](Game_correct.png)
![](Game_wrong.png)
![](table.png)

### 💰 Кнопка «Забрать деньги»
- Игрок может в любой момент нажать «Забрать деньги»
- Переходит на экран уровней, отображается сумма выигрыша

### 🏁 Финальный экран
- После завершения игры отображается экран:
  - Сумма выигрыша
  - Кнопки: **«Начать новую игру»** и **«Главное меню»**
